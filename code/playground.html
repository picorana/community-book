Load Network Data

Select a file with the following format:


    {
        "nodes" : [
            {   "id" : 0,
                "name": "A",
                "attributes": { "attr1": 1,
                                "attr2": 2,
                                "attr3": 3, ... }   },
            ... ],
        "edges" : [
            {   "source" : 0,
                "target": 1,
                "attributes": { "attr4": 4,
                                "attr5": 5, ... }   },
            ... ]
    }

The "source" and the "target" are ids of existing nodes and should differ from another.

The key "attributes" is optional. It contains a dictionary with the attribute labels and the corresponding numerical values. For nominal attributes you can use a constant value.
